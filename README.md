## 本工程使用说明：
- 先安装node然后全局安装gulp，bower,npm(npm一般安装node时就默认安装)
  
  1.安装gulp :

  ```bash
    $ npm install -g gulp 
  ```
  2.安装bower

  ```bash
    $ npm install -g bower
   ```
  3.运行 bower 自动下载所需第三方依赖（jquery，jquery-ui，angular)
  ``` bash
    $ bower install
    ```
